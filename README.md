# 🏦 Bank Loan Analysis Using Python

## 📌 Project Overview
This project focuses on analyzing bank loan data using Python to uncover customer behavior, loan performance, repayment trends, and financial risk patterns.

The analysis was performed using real-world business KPIs and interactive visualizations to identify insights related to loan quality, customer segmentation, regional trends, and repayment performance.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Reporting
- Business Insights
- Interactive Visualizations using Plotly
- Statistical Analysis using Pandas

---

## 🎯 Business Objectives

The primary objectives of this analysis were to:

- Analyze overall loan performance
- Identify good and bad loan trends
- Understand customer repayment behavior
- Evaluate regional loan distribution
- Track monthly lending growth
- Analyze financial risk indicators
- Generate business-focused insights from loan data

## 🔍  Key Insights

- Total loan applications processed exceeded **38K+** applications.
- Total funded loan amount reached approximately **$435.8M**.
- Total amount received from customers crossed **$473M**, indicating strong repayment performance.

### Loan Quality Insights
- Good loans accounted for **86.18%** of total applications.
- Bad loans represented only **13.82%** of the total portfolio.
- Good loans generated significantly higher repayment amounts compared to bad loans.

### Monthly Trend Insights
- Loan applications showed a consistent upward trend throughout the year.
- December recorded the highest loan activity with over **4,300 applications**.
- Funded amount and repayment trends peaked toward year-end, indicating business growth.

### Employment Insights
- Customers with longer employment history showed better repayment behavior.
- Borrowers with **10+ years** of employment received higher funded amounts.

### Loan Purpose Insights
- Debt consolidation loans contributed the highest share of funded amount.
- Credit card and home improvement loans were also major loan categories.

### Financial Risk Insights
- Higher DTI groups showed comparatively weaker repayment trends.
- Average borrower DTI remained around **13%**, indicating manageable debt levels overall.
